# Text-to-Speech

A simple, browser-based text-to-speech web app.

## Features

- Convert typed text into speech using the Web Speech API
- Adjustable voice, rate, pitch, and volume (if supported by the browser)
- Clean, minimal UI

## Tech Stack

- HTML
- CSS
- JavaScript (Web Speech API)

## Getting Started

1. Clone the repository:
   - [https://github.com/yugrajjoshi/Text-to-speech](https://github.com/yugrajjoshi/Text-to-speech)
2. Open index.html in your browser.

## Project Structure

- index.html — UI markup
- style.css — styling
- app.js — text-to-speech logic

## Usage

1. Enter text in the input area.
2. Select a voice and adjust settings if available.
3. Click the button to play speech.

## Browser Support

Modern browsers that implement the Web Speech API (for example, Chrome, Edge).
